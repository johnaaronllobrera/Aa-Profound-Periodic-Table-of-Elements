Project Overview
This project presents an interactive periodic table of elements. Each element is represented in a table cell, and clicking on an element displays additional information. The table includes categories such as Alkali Metals, Transition Metals, and more. The styling is applied through an external CSS file, and interactive behavior is controlled via JavaScript.

Features
Interactive Elements: Click on any element to display more information.
Categorization: Elements are categorized into groups like Alkali Metals, Transition Metals, etc.
Responsive Design: The table is styled to be visually appealing and functional.

Files Included
1. index.html: The main HTML file containing the structure of the periodic table.
2. style.css: The CSS file responsible for the styling of the periodic table.
3. iconic.png: The icon image used in the project.
4. Period.zip: A zip file containing a folder of photos associated with each element.

How to Run:
1. Download the files: Ensure you have `index.html`, `style.css`, `iconic.png`, and the `Period` folder in the same directory.
2. Open the HTML file: Double-click on `index.html` or open it in a web browser.
3. Interact with the Table: Click on any element to view more details.

Dependencies
- Web Browser: Most modern web browser (e.g., Chrome, Firefox, Safari).

Usage Instructions
1. Open `index.html` in a web browser.
2. Navigate through the periodic table.
3. Click on an element to display additional information about it.
4. Use the category buttons to filter and display elements by their group (e.g., Alkali Metals).

Customization
- Styling: Modify `style.css` to change the appearance of the periodic table.
- Elements Information: Update the HTML in `index.html` to modify or add new information for each element.
- Photos: Add or replace photos in the `period` folder to update the images associated with each element.
